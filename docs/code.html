code

The atom-code component is a simple way to render syntax highlighted code.

It allows passing multiple code-blocks, using atom-code-fragment, and letting the user toggle between them.

@Param: bind - the name of the state-variable to observe.

Example 1: Single

// #1 the following should render as formatted code const a = 1.0; // ok? function b() { return 1; } const c = () => { a() }

Example 2: Multiple

// #1 the following should render as formatted code const a = 1.0; // ok? function b() { return 1; } const c = () => { a() } // #2 the following should render as formatted code type Section = { html: string; lang: string; }; type HappinessScoreConfig = | number | (weatherOpinion: WeatherOpinion) => number; pre { border: collapse }
